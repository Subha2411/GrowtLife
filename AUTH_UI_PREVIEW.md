3. Enter email and password
4. Click "Create Account"
5. Modal closes
6. User email appears in top bar

### Returning User
1. Click "Sign In" button
2. Enter email and password
3. Click "Sign In"
4. Modal closes
5. User email appears in top bar
6. All previous logs appear (from Firestore)
7. User is logged in! âœ…

### Logout
1. Click "Logout" button in top bar
2. Modal closes
3. Sign In button appears again
4. Data remains in Firestore (safe)
5. Can log back in anytime âœ…

## What Happens Behind the Scenes

### When User Signs Up
1. Email/password sent to Firebase
2. Firebase creates user account
3. Auth state updated
4. Modal closes
5. Logs are synced from Firestore
6. User can start logging achievements

### When User Signs In
1. Email/password sent to Firebase
2. Firebase verifies credentials
3. Auth state updated
4. User data loaded from Firestore
5. All logs appear
6. Streak and companion level calculated

### When User Logs Out
1. Firebase signs out user
2. Auth state cleared
3. Local data remains (read-only mode)
4. User can sign in again anytime
